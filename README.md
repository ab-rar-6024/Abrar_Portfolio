## ✨ Features

- **Animated Loader** — Cinematic intro sequence on first load
- **Custom Cursor** — Glowing cursor with hover effects
- **Particle Background** — Interactive floating particles
- **Typing Effect** — Rotating role titles in the hero section
- **Scroll Animations** — Smooth reveal on scroll down and up
- **3D Tilt Cards** — Mouse-tracked perspective tilt on project cards
- **Theme Switcher** — 3 themes: Cyber (default), Light, Neon
- **Responsive Design** — Mobile-first, works on all screen sizes
- **Active Nav Highlight** — Navbar highlights current section on scroll
- **Scroll To Top FAB** — Floating button appears after scrolling down

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/ab-rar-6024/Abrar_Portfolio.git

# Navigate to project folder
cd Abrar_Portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
